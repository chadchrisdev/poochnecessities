How to Save These Avatars

To save these avatars for use in your app:

  1. Click the "Download PNG" button under each avatar
  2. Save the file to your assets/avatars/ directory
  3. Make sure to name them as avatar1.png, avatar2.png, and avatar3.png
  4. The avatars should now work with your placeholder avatar selection feature

Note: If the download buttons don't work in your environment, you can also:

  1. Right-click on each avatar
  2. Select "Save image as..."
  3. Save to your assets directory with the appropriate filename
